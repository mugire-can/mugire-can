## 🎨 Featured Projects

### 🎮 [Enclosehorse](https://github.com/mugire-can/enclosehorse)
A strategic puzzle game where players place walls on a grid to enclose the largest possible area with limited wall segments. Built with **TypeScript** and demonstrating game mechanics, grid systems, and algorithm optimization.

### 🎯 [MiniVibes - Wolfenstein 3D Engine](https://github.com/mugire-can/MiniVibes)
An educational 3D game engine implementing ray casting technology, inspired by classic Wolfenstein 3D. Features real-time 3D graphics rendering, game logic, collision detection, and performance optimization techniques.

### 🌐 [Portfolio Website](https://github.com/mugire-can/portofolio)
A modern, responsive portfolio showcasing cybersecurity expertise. Built with **HTML, CSS, JavaScript, and PHP**, featuring dynamic form handling with email notifications and 15+ professional animations.

### 🔐 [Ethical Hacking & Cybersecurity](https://github.com/mugire-can/ethical-hacking)
Comprehensive hands-on laboratory work covering OSINT (Open Source Intelligence) and penetration testing techniques with practical security research methodologies.

### 💻 [C Programming Runtrack](https://github.com/mugire-can/runtrack_c)
Comprehensive C programming project implementing string manipulation, pointers, arithmetic functions, struct handling, linked lists, and complete programs with file I/O.

### 🐳 [Docker & Virtualization](https://github.com/mugire-can/docker)
Educational exploration of Virtual Machines and Docker containerization—key technologies in modern server infrastructure and cloud computing.
